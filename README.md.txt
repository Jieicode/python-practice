# FX Price Tracker

## 概要
為替レート（USDJPY・EURJPY）をAPIから取得し、CSVに保存・分析するPythonプログラム。

## 機能
・為替レートをリアルタイム取得  
・CSVファイルに履歴を蓄積  
・前回との差分を計算  
・平均値・最高値・最安値を算出  
・移動平均（短期・長期）による売買シグナル判定  
・グラフ表示（USDJPY推移）

## 使用技術
・Python  
・requests（API取得）  
・csv（データ保存）  
・matplotlib（グラフ描画）

## 本日の進捗
・CSVへのデータ保存処理を実装  
・過去データの読み込み処理を実装  
・平均・最高値・最安値の計算機能を追加  
・移動平均（短期・長期）の計算を追加  
・買い / 売りシグナルの判定ロジックを実装  
・データ不足時の例外処理を追加
