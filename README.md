\## 作成プログラム



\## Number Game（数字当てゲーム）



Pythonで作成したコンソールゲームです。



ランダム生成された数字を回数制限内に当てるゲームです。



\### 機能



・メニュー画面  

・難易度選択  

・回数制限  

・ヒント表示（大きい/小さい）  

・再プレイ機能  

・遊び方表示
